About

Taamify classifies Torah cantillation marks (taamei hamikra) from audio. The production model is a five-block convolutional network trained on 64,517 verified word spectrograms from five readers, selected because it generalizes best to voices never heard in training (Leave-One-Reader-Out: 53% mean accuracy over four held-out readers, against a 9.1% chance baseline).

Labels were obtained by aligning Whisper transcriptions against the cantillated canonical text of the Sefaria API with exact-match, precision-first cleaning. Details, code and the full report are in the project repository.
